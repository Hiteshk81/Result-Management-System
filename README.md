## Features

Add student details like name, roll number, and marks

Save results locally without any backend

View all results only when you click Show Results

Clean and user-friendly interface

Built using proper HTML5 tags

## Technologies Used

HTML5

CSS3

JavaScript

## How to Run

Download the project folder

Open the index.html file in your browser

Add results using the form

Click Show Results to see all stored entries

## Notes

All data is stored in LocalStorage

Works fully offline
